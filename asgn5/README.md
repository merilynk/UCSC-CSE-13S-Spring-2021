# Assignment 5: Hamming Codes

These programs make up a Hamming(8,4) code generator and code decoder.

## Build
```
make
```
or
```
make all
```

## Running
```
./encode [-h] [-i infile] [-o outfile]
```
```
./decode [-h] [-v] [-i infile] [-o outfile]
```
Use -h to get the help/usage message.

## Cleaning
```
make clean
```
